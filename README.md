# Вариант 3 — Нечёткий поиск дубликатов товаров

Реализация алгоритма поиска дубликатов товарных предложений
на основе коэффициента Жаккара.

## Алгоритм
1. Предобработка текста (lowercase, удаление пунктуации)
2. Токенизация
3. Jaccard similarity
4. Фильтрация по порогу схожести

## Параметры (include/analyzer.h)
- SIMILARITY_THRESHOLD = 0.8
- USE_PREPROCESSING = true

## Язык
C++23 (используются std::ranges)

## Входные файлы
- catalog.txt
- new_items.txt

## Выход
- duplicates.json
